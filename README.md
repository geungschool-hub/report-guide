# 과학 탐구 보고서, 이렇게 쓴다

**주장–논거–고찰**을 중심축으로, 과학적 사실을 조사·해석해 자기 주장을 논거로 풀어내는
탐구 보고서 쓰는 법을 단계별로 익히는 학생용 인터랙티브 웹입니다.

- 7단계(주제·질문 → 주장·논거 → 자료 조사 → 개요 → 문단 → 고찰 → 서론·결론) + 인용·출처 / 보고서 구조 / 자기점검 체크리스트 / 내 계획서
- 각 단계 = 핵심 개념 + 좋은👍/나쁜👎 예시 대비 + 꿀팁 + ✏️ 직접 해보기(자동 저장)
- 계획서 단계에서 **PDF 내보내기** · 전체 복사 지원
- 설치·서버·인터넷 없이 브라우저에서 바로 동작(단일 HTML), 작성 내용은 브라우저 localStorage에 자동 저장
